### 3. 답변 제출 방식

> **기한: 퀴즈 출제일 기준, 다음 주 화요일 오후 6시까지**
> 
> - Ex) 1주차: 4월 5일 낮 12시 ~ 4월 11일 오후 6시

> **미제출 혹은 지각 시 3,000원의 벌금 납부**
> 
> - 후에 있을 프로미스 종강 총회에서 사용할 예정
> - **3회 이상 시 소모임에서 탈퇴 처리**가 되니, 모두들 유의해주세요☠️
